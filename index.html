Which AI actually solves the captcha — cheapest?

Every candidate model run over the same saved puzzles, scored on the only thing that counts: clicking the right little drawings in the right order. Ranked by real solve-rate, then speed and cost. Target: free / ≤ $0.50 per 1,000 solves.

What is this?

We scrape the Smyths toy catalogue through Tor (free, anonymous IPs). Now and then the site throws up a security puzzle from hCaptcha — the doodle kind: a photo with a few faint hand-drawn icons scribbled on it, and a strip up top saying “select in this order.” You have to click the doodles in that exact sequence.

To get past it automatically you need a vision model that can (1) find the doodles and (2) put them in the right order. This page is the bake-off: same real puzzles, every model, head-to-head — so we can pick the one that solves it reliably for the least money. See where each model actually clicked →

Exit health — the bigger lever

The puzzle only appears on low-trust connections. Most fetches never see one — so the real win is avoiding captchas, not solving them faster.
